# Zillow Data
This repo contains the forecasting for a time series data using python and ARIMA models.<br/>
Precise analysis has been made to the data, including but not limit to, checking and addressing missing values and seasonality detection.<br/>
Finally a visualization story is created for this data only on 5 states using Tableau.<br/>
